# [tenant-eviction-skill]

## What This Skill Does

[Helps a tenant who received a 14-day eviction notice understand their rights, timeline, and options. Guides them through the process step bystep, using plain language. Intended user is a non-lawyer.]

## How to Use

1. Download the `SKILL.md` file from this repository
2. Go to claude.ai → Settings → Skills
3. Upload the `SKILL.md` file
4. Toggle the skill on
5. Start a new conversation and try one of the sample prompts below

## Sample Prompts

Try these prompts to test the skill:

- "[I got evicted. What should I do?]"
- "[Rights and options after getting evicted.]"
